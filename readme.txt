== Installation ==

1. Upload the plugin to `/wp-content/plugins` and activate. OR Search "Wp Live Chat" from `WP Dashbard --> Plugins --> Add New` then hit Install-- Then activate the plugin

2. Access the plugin from `Dashboard --> Settings --> Wp Live Chat. 

3. To get your code you must register at [https://sellper.net](https://sellper.net) , Click on Try Nnow, Fill the form, you will receive your code by mail.

4. Paste your code from Sellper chat and click save.
